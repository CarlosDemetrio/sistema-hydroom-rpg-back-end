# T1.1 - Configuration DTOs

## Objective
Create request/response/update DTOs for each configuration entity with `ValidationMessages`.

## Scope
Configuration entities: Atributo, Aptidao, Bonus, Classe, DadoProspeccao, Genero, Indole, MembroCorpo, Nivel, Presenca, Raca, TipoAptidao, Vantagem.

## Steps
1. Define record-based DTOs in `dto/request` and `dto/response`.
2. Apply validation annotations using `ValidationMessages`.
3. Ensure response DTOs omit internal/audit fields.

## Formula Standardization Prep
- Define formula variable naming rules (ALL CAPS, underscore for multi-word).
- Document allowed functions/operators for user guidance.
- Ensure config DTOs include validation hooks for formula fields (validator will be added later).

## Acceptance Checks
- DTOs exist for all config entities.
- All validations use `ValidationMessages` constants.
