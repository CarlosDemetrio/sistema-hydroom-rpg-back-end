# T8.1 - DTOs as Records

## Objective
Convert request/response DTOs to Java records where feasible.

## Steps
1. Inventory DTO classes and identify record candidates.
2. Convert to records and adjust constructors/usages.
3. Ensure validation annotations remain on record components.

## Acceptance Checks
- DTOs are records unless a class is required.
- No functional regression in serialization.
