# T13.9 - CloudWatch Logging Configuration

## Objective
Configure CloudWatch Agent with minimal retention and WARN-level framework logs.

## Steps
1. Configure CloudWatch Agent to read local log file.
2. Set retention to 3 days.
3. Adjust log levels to WARN for frameworks.

## Acceptance Checks
- Logs ship to CloudWatch with 3-day retention.
- Framework logs are WARN or higher.
