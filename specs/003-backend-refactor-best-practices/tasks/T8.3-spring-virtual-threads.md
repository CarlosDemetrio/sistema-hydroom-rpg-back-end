# T8.3 - Enable Virtual Threads and ProblemDetails

## Objective
Enable Spring Boot 4 features for low resource usage.

## Steps
1. Set `spring.threads.virtual.enabled=true` in production config.
2. Set `spring.mvc.problemdetails.enabled=true` in production config.
3. Validate behavior in dev/staging if available.

## Acceptance Checks
- Virtual threads and ProblemDetails are enabled in prod profile.
- No startup errors related to these settings.
