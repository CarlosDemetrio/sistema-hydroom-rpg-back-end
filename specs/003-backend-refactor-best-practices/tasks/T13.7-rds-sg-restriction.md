# T13.7 - RDS Security Group Restriction

## Objective
Restrict RDS access to EC2 security group only.

## Steps
1. Update RDS SG to allow only EC2 SG on 5432.
2. Ensure RDS is not publicly accessible.
3. Validate connectivity from EC2.

## Acceptance Checks
- RDS SG allows only EC2 SG.
- RDS remains private.
