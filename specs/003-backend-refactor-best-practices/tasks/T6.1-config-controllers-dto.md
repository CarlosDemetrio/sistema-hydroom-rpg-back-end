# T6.1 - Configuration Controllers DTO Refactor

## Objective
Refactor configuration controllers to use DTOs and mappers only.

## Steps
1. Replace entity request/response types with DTOs.
2. Apply `@Valid` to request DTOs.
3. Use mappers for entity conversions in controllers.

## Acceptance Checks
- No configuration controller exposes entities.
- All config endpoints use DTOs + mappers.
