# T6.3 - Security Annotation Verification

## Objective
Verify that all endpoints in the application have the appropriate security annotations to protect against unauthorized access.

## Steps
1. Review the code in the `src/main/java/br/com/hydroom/rpg/fichacontrolador/controller/` directory.
2. Check the `SecurityConfig.java` file in the `src/main/java/br/com/hydroom/rpg/fichacontrolador/config/` directory.
3. Ensure that all endpoints have the necessary security annotations, such as `@PreAuthorize`, `@Secured`, or `@RolesAllowed`.
4. Verify that the security configuration is correctly set up to require authentication and authorization for protected resources.

## Acceptance
- All endpoints are protected with the appropriate security annotations.
- The security configuration is correctly set up and applied.
