# T5.2 - Mapper Usage Enforcement

## Objective
Ensure DTO mapping happens only in controllers and not in services.

## Steps
1. Scan services for DTO mapping or mapper usage.
2. Move mapping logic to controllers.
3. Ensure services only accept/return entities.

## Acceptance Checks
- No mapper usage in services.
- Controllers handle DTO conversion consistently.
