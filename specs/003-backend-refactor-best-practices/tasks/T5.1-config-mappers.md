# T5.1 - Configuration Mappers

## Objective
Create mapper classes for each configuration entity.

## Steps
1. Create mapper per config entity in `mapper/`.
2. Map entity -> response DTO and request DTO -> entity.
3. Keep mappers used only in controllers.

## Acceptance Checks
- Mapper exists for each config entity.
- Controllers use mappers; services do not.
